Welcome to the Delightful Eatery API!

Explore our delicious cuisines and eateries through our API endpoints.

GET x/api/eatery - Fetch all entries

GET/:id /api/eatery/:id - Fetch a specific entry

POST /api/eatery - Create a new entry

PUT /api/eatery/:id - Update an existing entry

DELETE /api/eatery/:id - Delete an existing entry
